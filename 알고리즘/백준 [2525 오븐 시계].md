# 백준 [2525 오븐 시계]
```python
H, M = map(int, input().split())    #시간과 분
T = int(input())    #요리하는 데 필요한 시간(분)
H += T//60          
M += T%60           
if H >= 24: #더했을 때 시간이 24시간이 넘으면
    H = H%24    #24로 나눈 나머지가 시간이 된다.
if M >= 60: #더했을 때 분이 60분이 넘으면
    H += M//60  #60으로 나눈 몫이 시간이 된다.
    M = M%60    #60으로 나눈 나머지가 분이 된다.
if H >= 24: #더했을 때 시간이 24시간이 넘으면
    H = H%24    #24로 나눈 나머지가 시간이 된다.
print(H,M)

'''
내가 한 풀이지만 너무 지저분해서 더 생각해보니 분을 먼저하고
시간을 나중에 하면 if문 하나를 줄일 수 있었다.
'''
H, M = map(int, input().split())    #시간과 분
T = int(input())    #요리하는 데 필요한 시간(분)
H += T//60          
M += T%60           
if M >= 60: #더했을 때 분이 60분이 넘으면
    H += M//60  #60으로 나눈 몫이 시간이 된다.
    M = M%60    #60으로 나눈 나머지가 분이 된다.
if H >= 24: #더했을 때 시간이 24시간이 넘으면
    H = H%24    #24로 나눈 나머지가 시간이 된다.
print(H,M)
```
궁금해서 다른 분들의 풀이를 봤다. 나랑 비슷하지만 더 쉽게 푸셨다. 내가 어렵게 생각했다보다.
```python
H, M = map(int, input().split())
timer = int(input()) 
H += timer // 60
M += timer % 60
if M >= 60:
    H += 1
    M -= 60
if H >= 24:
    H -= 24
print(H,M)
'''
방식은 똑같고 if문에서 차이가 있는데...if문 위에 60으로 나눈 몫과 나머지를 각각 시간과 분에 더했을때
이미 시간과 분이 각각 120분 48시간을 넘지 못하기 때문에 분이 60분 넘으면 그냥 시간에 1을 더하고 분에 60을 빼면 됐다.
마찬가지로 시간이 24시가 넘으면 그냥 시간을 24시간 빼주면 됐다.
물론 순서는 분 먼저 시간이 나중에 이렇게 해야 중복되서 짧게 쓸 수 있다.
'''
