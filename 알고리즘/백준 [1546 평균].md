# 백준 [1546 평균]
```python
N = int(input())    #과목의 개수를 받는다.
L = list(map(int, input().split())) #과목에 대한 점수를 리스트로 받자
M = max(L)      #리스트에서 최대값
sum = sum(L)    #리스트의 총합
avg = sum * 100 / (M * N)   #평균
print(avg)
'''
만약 N이 3이라고 하면
점수는 a, b, c 이라고 하면
평균은 (a + b + c)/N 이다.
a, b, c 중에서 최댓값을 M 이라고 하면
점수를 바꾸는 방법이 a -> a/M*100인데 b와 c도 같은 방법으로 점수를 바꾸면
평균은 (a + b + c)/N에 최댓값을 나누고 100을 곱하는 것과 같다.
따라서 (a + b + c) * 100 / (N * M) 이다.
'''
```
