3. reverse 함수를 사용한다.
   입력받은 문자열을 list로 변환시켜주고 list 내의 요소를 역순으로 정렬한다.
   join 함수를 사용하여 문자열로 변환 후 출력한다.
   ```python
   test = 'hello'
   test_list = list(test)

   print(test_list)  #['o','l','l','e','h']

   ```
