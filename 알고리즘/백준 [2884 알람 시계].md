# 백준 [2884 알람 시계]
```python
a, b = map(int, input().split())    #시간(시, 분)을 받는다.
if b>=45:           #분이 45분보다 크면
    print(a, b-45)  #45분 뺀 만큼만 그냥 보여주면 된다.
else :              #분이 45분보다 작으면
    a -= 1          #시간은 한시간 줄고
    b = 15+b        #분도 바뀐다.
    if a < 0:       #원래 시간은 0시 여서 음수가 되는경우 
        print(23, b)    #23시로 바꾸고 분도 바뀐 분으로 출력
    else:
        print(a, b)
```
