# 백준 [9506 약수들의 합]
```python
while 1:                #무한 루프
    n = int(input())    #정수를 입력 받는다.
    if n == -1:         #받은 정수가 -1이면
        break           #종료!
    arr = []            
    for i in range(1, n):   #1부터 n-1까지 반복
        if n%i == 0:        #i가 n의 약수인지 확인
            arr.append(i)   #약수이면 arr 리스트에 추가
    if sum(arr) == n:       #리스트에 추가한 원소를 다 더했을 때, 처음 받은 정수이면 완전수!!!
        print(n, " = ", " + ".join(str(i) for i in arr), sep = "")
    else :                  #완전수가 아니다..
        print(f"{n} is NOT perfect.")
```
```
위의 작성한 코드에서
print(n, " = ", " + ".join(str(i) for i in arr), sep = "")
이 부분을 작성하지 못해서 실패가 떴었다.

하나씩 해석해보려고 한다.
1. 'n' : 완전수인 수를 출력한다.
2. ' " = " ' : 완전수와 완전수의 약수들을 구분하기 위한 문자열
3. ' " + ".join(str(i) for i in arr) ' : 이 부분은 'arr' 리스트의 각 요소를 문자열로 변화하여
  "+" 문자로 연결한다. 즉, 'arr' 리스트의 각 요소를 문자열로 변환하여 "+"로 연결한 후 하나의 문자열로 만든다.
  예를 들어, 'arr' 리스트가 [1, 2, 3]이라면, 이 부분은 "1 + 2 + 3"으로 변환된다.
4. 'sep = ""' : sep 매개변수는 출력되는 요소들 사이에 넣을 구분자를 지정한다. 여기서는 구분자 없이 출력되도록 빈 문자열로 설정.
```
