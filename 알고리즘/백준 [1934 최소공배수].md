# 백준 [1934 최소공배수]
```python
def gcd(a,b):       #최대공약수를 정의한다.
    while b:        #b가 0이 아니면 계속 반복
        a, b = b, a%b   #유클리드 호제법
    return a        #최대공약수
T = int(input())    #테스트 케이스 수
for _ in range(T):  #테스트 케이스 수만큼 반복
    a, b = map(int, input().split())    #두 수를 받는다.
    print(int(a*b/gcd(a,b)))    #두 수의 곱의 최대 공약수를 나누면 최소공배수가 된다.


'''
<유클리드 호제법>
숫자 a, b가 있을 때,
a를 b로 나눈 나머지와 b의 최대공약수는
a와 b의 최대공약수와 같다.

계속해서 a를 b로 나누어서 b를 a에 나눈 나머지를 b에 대입시켜서
b가 0이 될때까지 반복을 하면,
남는 a 값이 바로 최대공약수이다.

cf. L(최소공배수) , G(최대공약수)일 때
L * G = a * b
'''
```
