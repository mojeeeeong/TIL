# 백준 [2530 인공지능 시계]
```python
#나의 풀이인데 백준에서 오류라고 뜬다.
H, M, S = map(int, input().split()) #시, 분, 초
T = int(input())    #요리하는데 필요한 시간(초)
H += T//3600
M += (T%3600)//60
S += (T%3600)%60
if S >= 60:
    S -= 60
    M += 1
if M >= 60:
    M -= 60
    H += 1
if H >= 24:
    H -= 24
print(H, M, S)
```
