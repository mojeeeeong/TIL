# 백준 [10039 평균 점수]
```python
sum = 0     #점수의 총합을 sum이라 하고 0으로 초기화
for _ in range(5):  #5명의 점수를 반복문으로 받자
    A = int(input())    #각각 점수를 정수로 받는다.
    if A < 40:          #점수가 40점 미만이면
        A = 40          #40점으로 간주
        sum += 40       #sum에 40을 더한다.
    else:               #점수가 40점 이상이면
        sum += A        #받은 점수를 sum에 더한다.
print(int(sum/5))       #총합을 5로 나눠 평균을 내고 정수로 변환하여 출력
```
